リスト

リストは配列に非常によく似ています。リストは任意のタイプの変数を含むことができ、任意の数の変数を含むことができます。リストは非常に簡単な方法で反復処理することもできます。以下は、リストを構築する方法の例です。

```python mylist = [] mylist.append(1) mylist.append(2) mylist.append(3) print(mylist[0]) # prints 1 print(mylist[1]) # prints 2 print(mylist[2]) # prints 3

prints out 1,2,3

for x in mylist: print(x) ```

存在しないインデックスにアクセスしようとすると、例外（エラー）が発生します。

python mylist = [1,2,3] print(mylist[10])
