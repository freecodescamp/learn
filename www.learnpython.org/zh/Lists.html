列表

Lists 与 arrays 非常相似。它们可以包含任何类型的变量，并且可以包含任意数量的变量。Lists 也可以非常简单地进行迭代。这里是一个如何构建 List 的示例。

mylist = []
mylist.append(1)
mylist.append(2)
mylist.append(3)
print(mylist[0]) # prints 1
print(mylist[1]) # prints 2
print(mylist[2]) # prints 3

# prints out 1,2,3
for x in mylist:
    print(x)

访问一个不存在的索引会生成一个异常（错误）。

mylist = [1,2,3]
print(mylist[10])
